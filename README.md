# 🚗 Middle Man Motors

> **Safe, Reliable, and Affordable Vehicle Platform**  
> Serving customers from Georgia to Virginia with transparent, honest car buying—no hidden fees, no pressure.

---

## 🛠️ Tech Stack & Architecture

This project is built using a modern, ultra-fast, serverless stack designed for zero hosting overhead and instant global deployment.

* **Frontend:** React (Vite) + TypeScript + Tailwind CSS
* **UI Components:** Shadcn UI + Lucide Icons
* **Database & Auth:** Supabase (PostgreSQL, Row Level Security, Auth)
* **Hosting & CDN:** Cloudflare Pages (Continuous Integration via GitHub)
* **State Management:** Zustand (Vehicle Comparison & Cart State)

---

## ⚡ Key Features

* 🚘 **Dynamic Vehicle Inventory:** Filter, search, and view detailed specs for available cars.
* ⚖️ **Vehicle Comparison Tool:** Compare side-by-side specs, pricing, and features.
* 📝 **Fast Financing:** Integrated 60-second Pre-Qualification and direct Credit Applications.
* 🛍️ **Merch Store & Cart:** Built-in shopping drawer for dealership merchandise.
* 🏛️ **Government Contracting:** Dedicated portal for municipal and corporate vehicle supply.
